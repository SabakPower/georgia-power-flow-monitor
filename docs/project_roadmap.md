# Project Roadmap

## Georgia Power Flow Monitor

This project will be developed step by step as an open-source energy data and dashboard project focused on Georgia's electricity system.

## Phase 0 — Setup

Goal:
Create the technical foundation for the project.

Tasks:
- Create GitHub repository
- Set up Python environment
- Install required packages
- Create repository folder structure
- Add README and starter documentation

Status:
In progress.

## Phase 1 — Data Source Audit

Goal:
Identify and document public electricity data sources.

Tasks:
- Map available data from ESCO, GSE, GENEX, GNERC and secondary sources
- Create a data source catalogue
- Document source quality, frequency, format and access method
- Decide which source to use first

Status:
In progress.

## Phase 2 — First Manual Dataset

Goal:
Create the first clean dataset manually from one official source.

Planned tasks:
- Download one electricity balance or generation/consumption file
- Save original file in data/raw
- Extract relevant table manually
- Create cleaned CSV in data/processed
- Document assumptions and transformations

Status:
Not started.

## Phase 3 — First Analysis

Goal:
Use Python to load and analyse the first dataset.

Planned tasks:
- Load CSV with pandas
- Clean columns and units
- Create first chart
- Export chart to outputs/charts
- Write short analysis note

Status:
Not started.

## Phase 4 — Basic Dashboard

Goal:
Create a simple Streamlit dashboard.

Planned tasks:
- Add file loading
- Add basic charts
- Add source notes
- Add simple navigation

Status:
Not started.

## Phase 5 — Public Communication

Goal:
Use the project for GitHub and LinkedIn visibility.

Planned tasks:
- Publish GitHub updates
- Write LinkedIn build-in-public posts
- Share project milestones
- Connect technical work to power markets, renewables and grid analysis

Status:
Not started.
