# Georgia Power Flow Monitor
Georgia Power Flow Monitor is an open-source energy data project focused on Georgia's electricity system.

The goal is to collect, clean, analyse and visualise public electricity data from Georgian institutions such as ESCO, GSE, GENEX, GNERC and other relevant sources.

## Project Objectives
- Track electricity generation, consumption, imports, exports and system balance
- Build a public source catalogue for Georgian electricity data
- Create clean datasets from public reports and downloadable files
- Develop a Streamlit dashboard for system-level electricity monitoring
- Share progress through GitHub and LinkedIn as a public portfolio project

## Current Phase
Phase 0-1: project setup and data source audit.

## Planned Data Sources
- ESCO — electricity balance, settlement indicators and market-related data
- GSE — power system, generation, consumption and grid data
- GENEX — day-ahead and intraday market information
- GNERC — regulatory and electricity market reports
- Secondary sources — market reports and analytical publications
